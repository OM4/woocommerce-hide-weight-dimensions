Ensures that product weight and dimensions aren't displayed on the Additional Information tab when viewing a WooCommerce product.

== Description ==

WooCommerce no longer has an option to disable displaying of product weight and dimensions information.

Activate this plugin, and the additional information tab will no longer include Weight and Dimensions!

If you have any other suggestions for other information that could be included, please [add your suggestion via GitHub](https://github.com/OM4/woocommerce-hide-weight-dimensions/issues).

== Installation ==

1. Download the latest stable version of the plugin from https://github.com/OM4/woocommerce-hide-weight-dimensions/archive/release.zip.
1. Install this plugin via WordPress Dashboard, Plugins, Add New, Upload.
